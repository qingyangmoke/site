Methods

wrap(sprite, paddingopt, useBoundsopt, horizontalopt, verticalopt)

Source:
  • WorldWrap.js, line 1
该方法会检查给定的游戏对象的x、y坐标是否在游戏边界之内。 如果超出游戏边界，会将对象重新定位到边界的对面，实现一个对象在游戏中环绕运动的效果。 如果精灵有一个物理Body，那么（sprite.body）应该作为第一个参数传递给函数。 这个方法也有局限性，例如，如果你缩放了游戏容器，然而对象不会总是被正确重新定位，这时候你就需要使用自己的wrap函数了。
Parameters:
Name Type Attributes Default Description
sprite Tiny.Sprite | Tiny.Image | Tiny.TileSprite | Tiny.Text 游戏对象
padding number <optional>
 0 x,y的空余区域，如果设置了useBounds 该参数会被忽略
useBounds boolean <optional>
 false If useBounds is false wrap checks the object.x/y coordinates. If true it does a more accurate bounds check, which is more expensive.
horizontal boolean <optional>
 true 是否开启横向wrap 默认开启
vertical boolean <optional>
 true 是否开启纵向wrap 默认开启
