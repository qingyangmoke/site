Rope

.mesh. Rope

The rope allows you to draw a texture across several points and them manipulate these points ```js for (let i = 0; i < 20; i++) { points.push(new Tiny.Point(i * 50, 0)); }; var rope = new Tiny.mesh.Rope(Tiny.Texture.fromImage('snake.png'), points); ```
