样式

设置 style 可以更改文字的样式：

txt.style = {
  fill: '0xff0000,
  fontFamily: 'PetMe64',
  fontSize: 16
};

其他样式属性像 stroke 设置字体描边颜色，strokeThickness 设置描边粗细，dropShadow 设为 true 文字就有了阴影，dropShadowColor 让阴影换个16进制的颜色，dropShadowDistance 让阴影的距离变化一下。
也可以设置 wordWrap 为 true 让大段文字换行，当然要设置 wordWrapWidth 想要换行的最大宽度，align 可以设置多行文字的对其方式。

txt.style = {
  wordWrap: true,
  wordWrapWidth: 100,
  align: 'center'
};

注意

align 不适应于单行文字

如果你想要使用外部字体文件，使用 CSS 的 @font-face 规则，页面加载后，就可以设置文字的 fontFamily 来使用。

@font-face {
  font-family: "fontFamilyName";
  src: url("fonts/fontFile.ttf");
}
txt.style = {
  fontFamily: 'fontFamilyName'
};

别忘了在你的 CSS 文件里添加 font-face 规则。
