简单使用

使用 Text 类可以创建字体，需要传入两个参数：字符和文字样式。下面创建字符是"你好，Tiny.js！"，样式是白色、32像素：

var txt = new Tiny.Text('你好，Tiny.js！', {
  fontFamily: 'Arial',
  fontSize: 32,
  fill: 'white'
});

由于 Text 类继承自 Sprite，所以它拥有像 x，y，width，height，alpha 和 rotation 等属性。设置位置和尺寸就像操作精灵一样。
如果你想要在创建之后再更改文字的内容，设置 text 属性即可：

txt.text = 'Text changed!';
