定时器

使用方法 countDown 便可以快速创建一个定时器：

var ticker = new Tiny.ticker.Ticker();
ticker.countDown({
  duration: 3e3,
  callback: function () {
    console.log('每三秒打印一次');
  }
});
ticker.start();

或使用 Ticker 的单例：

var ticker = Tiny.ticker.shared;
ticker.countDown({
  duration: 3e3,
  callback: function () {
    console.log('每三秒打印一次');
  }
});
// 单例在主调度中已经 start，故不用再执行 start()

入参

  • duration：控制每次的间隔，单位是 ms。
  • times：次数，不传即无限次
  • callback：每次间隔的回调
  • complete：达到总次数后的回调
ticker.countDown({
  // 6 秒一次
  duration: 6e3,
  // 共 2 次
  times: 2,
  // 每 6 秒调用一次
  callback: function() {
    console.log('go on.');
  },
  // 6*2 秒后调用，完成
  complete: function () {
    console.log('time out.');
  },
});
