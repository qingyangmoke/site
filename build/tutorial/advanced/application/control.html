游戏控制

游戏整体的控制比较重要，当然，使用也很简单。

让游戏停止：

app.stop();

开始：

app.start();

暂停：

app.pause();

继续：

app.resume();

注意
⚠️⚠️⚠️ 这很重要，为避免移动端压后台引发的运行 Crash，需要你控制一下游戏，如：

// 页面压后台，让游戏停下来
document.addEventListener('pause', function (e) {
  app.pause();
}, false);

// 页面恢复运行，让游戏继续
document.addEventListener('resume', function (e) {
  app.resume();
}, false);
