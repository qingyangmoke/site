场景切换

如前所述，使用 Application 的实例化对象 app 的 replaceScene 方法来切换场景：

// 以 MoveInL 方式切换到场景 sceneName
app.replaceScene(sceneName, 'MoveInL');

app 的 replaceScene 方法接受 >= 3 个参数：

replaceScene(scene[DisplayObject], transition[String], duration[Number], ..);

其中第4个参数是按需传入的，比如 FadeColor 需要传入16进制颜色值：

app.replaceScene(sceneName, 'FadeColor', 500, 0xFF0000);

你也可以直接通过 Transition 转场类操作两个显示实例之间的转场：

var trans = new Tiny.Transition(scene1, scene2, duration);
trans['FadeColor'](0xFF0000);

更多切换动画请转到“转场”篇。
