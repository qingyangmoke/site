键盘

插件名：Keyboard
npm：tinyjs-plugin-keyboard

你可以使用 Tiny.Keyboard 类初始化一个键盘控制对象，然后设置"按下（press）"和"松开（release）"的回调监听。
现在，我们设置一下空格键的触发：

var keySpace = new Tiny.Keyboard(32);
keySpace.press = function (e) {
   console.log('space pressed!')
}
keySpace.release = function (e) {
   console.log('space released!')
}
