滚动

插件名：Scroller
npm：tinyjs-plugin-scroller

Tiny.js 使用了 scroller，你可以通过 Tiny.Scroller 获得，举个栗子：

var scrollerObj = new Tiny.Scroller(function(left, top, zoom) {
    // apply coordinates/zooming
}, {
    scrollingY: false
});

如果需要某个显示对象支持滚动，可以使用 Tiny.Layer 类来创建：

var ant = Tiny.Sprite.fromImage('https://zos.alipayobjects.com/rmsportal/IJwbZnAkuJOhKqDXngUm.png');
var layer = new Tiny.Layer();
layer.setScroll(Tiny.WIN_SIZE.width, Tiny.WIN_SIZE.height);
layer.setHitArea(new Tiny.Rectangle(0, 0, Tiny.WIN_SIZE.width, Tiny.WIN_SIZE.height));
layer.container.addChild(ant);
app.run(layer);
