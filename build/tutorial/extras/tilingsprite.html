TilingSprite

插件名：tiling
npm：tinyjs-plugin-tiling

平铺某个纹理，一般用于制作游戏中的地图，比较简单，直接看代码片段吧：

var texture = Tiny.Texture.fromImage('https://zos.alipayobjects.com/rmsportal/qphRqMulIoaBUPQnfEBV.png');
var tilingSprite = new Tiny.tiling.TilingSprite(texture, Tiny.WIN_SIZE.width, Tiny.WIN_SIZE.height);
var container = new Tiny.Container();
container.addChild(tilingSprite);
app.run(container);
